## TeXt 테마 사용 관련

### 
- 기본 포스트 설정 위치는?
  - `_config.yml` 의 `defaults` 부분
    ```yml
    defaults:
    - scope:
        path: ""
        type: posts
        values:
        layout: article
        sharing: true
        license: true
        aside:
            toc: true
        show_edit_on_github: true
        show_subscribe: true
        pageview: true
    ```
