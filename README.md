## blog 운영 관리

### 1. 메뉴 생성

- `_config.yml`에서 `navigation`설정 값 추가

  - 예시

    ```yml
    navigation:
      - name : tag # 표시될 이름
        url : /tag # 링크 주소 값
    ```

- 새로운 메뉴 추가 시 체크 사항
  - [ ] 해당 메뉴 명 디렉토리 생성
  - [ ] 디렉토리 내 해당 메뉴명 마크업 생성(`.md`)
  - [ ] 메뉴별 레이아웃 추가 필요 시 생성(`_layout` 하위)
